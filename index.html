Jungle Runners

By: Sinchana, Chloe, Claire and Samuel


ABOUT THE GAME


Jungle Runners is a remake of the game: Fireboy and Watergirl.
This is a 2 player game where both players work together to collect gems while completing the map before time runs out.

POSSIBLE CHALLENGES



1. String Functions

We will be using string functions to ask the player for their names and convert their name to all upper case. They will also be used to display the score, time and leader boards. This is done by concating strings and number values into a single string.



2. Arrays

We will use arrays to store values for each player and assets that are a part of the game i.e. sprites, animations, etc...



3. Bubble Sorting

We will be using bubble sorting to display the leaderboard after each level/map. The leader board will have the time it took for players to complete the map as well as points based on how many gems they collected.

























SCREENS


The following screens will be present in our game:

1. Starting Scene - This scene will have give players the option to either start the game or read the intructions before beginning.

2. Main Lobby Scene - This is where the different levels/maps are displayed. There are 2 levels/maps.

3. Map #1 Scene - This is where players can play on map 1.

4. Map #2 Scene - This is where players can play on map 2.

5. End Scene - This is where the leader board is shown after players complete their map.
