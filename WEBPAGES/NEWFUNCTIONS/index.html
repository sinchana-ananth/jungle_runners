NEW FUNCTIONS

About


This page contains some of the new functions we used while creating our game!



Function #1 : destroy()

The function, destroy(), removes an object from the game.
We used this function to remove the purple gem after character 1 collects (touches) it. When character 1 touches this gem, it is removed from the screen.



Function #2 : setTint(colour)

The function, setTint(), is used to set a tint over an object.
We used this function to set a dark tint over the play button on the start screen. When the user clicks on the button, it will become slightly darker.



Function #3 : clearTint(colour)

The function, clearTint(), is used to clear a tint over an object.
We used this function to clear the dark tint from the play button on the start screen. When the user takes their cursor away from the button, it clears the tint and restores the button's original colour.



Function #4 : setShadow(vert, horiz, color, blur, shadowStroke, shadowFill)

The function, setShadow(), sets a drop shadow behind an object. It typically has 6 parameters. (1) The horizontal shadow offset. (2) The vertical shadow offset. (3) Shadow colour. (4) Blur radius. (5) Stroke/no stroke. (6) Fill/no fill.
We used this function to set a shadow behind the text on the start screen so that it stands out.



Function #5 : setTimeout(function, delay)

The function, setTimeout(),sets a timeout for a specific amount of time in milliseconds.
We used this function to animate our title in the start screen. This delays the recursive function.
